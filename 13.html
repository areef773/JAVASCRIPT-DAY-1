Difference between == and === in JavaScript == (Loose Equality) Compares values after type conversion Automatically converts data types if they are different Example: 5 == "5" // true (string is converted to number) === (Strict Equality) Compares both value and data type No type conversion happens Example: 5 === "5" // false (number ≠ string) In short: == → checks value only (with conversion) === → checks value + type (safer and recommended)
